Descripción:

La aplicación permite a los usuarios realizar compras seleccionando productos de una tienda virtual. Los usuarios pueden agregar productos al carrito, confirmar la compra y gestionar su sesión. Además, incluye funcionalidades de registro de usuarios y un panel administrativo para gestionar usuarios y productos.
Bibliotecas Utilizadas

•	SweetAlert2: Para mostrar mensajes modales interactivos.
•	Bootstrap: Framework de CSS para el diseño responsivo y componentes.
•	jQuery: Biblioteca JavaScript para manipulación del DOM y eventos.
•	Toastify: Para notificaciones estilo toast.
•	Dog CEO API: API utilizada para obtener imágenes de perros (en la página "Nosotros").

Archivos Principales

•	store.js: Funcionalidades relacionadas con el carrito de compras y almacenamiento local.
•	sesion.js: Gestión de sesiones de usuario y redirección según el estado de sesión.
•	scart.js: Carga y muestra los productos confirmados para la compra.
•	register.js: Registro de nuevos usuarios y creación de usuario administrador por defecto.
•	panel.js: Panel administrativo para gestionar usuarios registrados y asignar roles de administrador.
•	nosotros.js: Utiliza una API para obtener imágenes de perros y maneja el formulario de contacto.
•	login.js: Inicio de sesión de usuarios y gestión del estado de sesión.
•	tienda.html: Página principal de la tienda con funcionalidades de búsqueda y gestión de productos.
•	Otros archivos: Estilos CSS, imágenes y videos utilizados en el proyecto.
